# 抽象相亲模拟器

一个轻松搞笑的相亲模拟游戏。

## 玩法

1. 点击「开始相亲」进入游戏
2. 观看每位相亲对象的开场表演
3. 从3个选项中选择你的回复
4. 观察对方的反应和好感度变化
5. 看完10轮对话，触发不同结局

## 角色

游戏包含9位风格迥异的相亲对象：

- 币圈陈总 - 只差你一个的区块链大佬
- 等待你的我 - 普通且迷之自信的优质青年
- 成熟稳重懂生活 - 年纪大=会疼人
- 听妈妈的话 - 妈妈说系列的经典代表
- 灵魂在流浪 - 追求精神自由的诗人
- 猛男本男 - 健身教练の迷惑行为
- 代码敲得不咋样 - 真诚老实的程序员
- 梗比人红 - 脱口秀演员的相亲之路
- 搬砖也要优雅 - 踏实靠谱的建筑工人

## 技术

纯单HTML文件，无需任何依赖，直接用浏览器打开index.html即可运行。
